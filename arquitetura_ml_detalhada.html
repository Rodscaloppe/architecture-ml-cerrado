⚙️ 2. PRÉ-PROCESSAMENTO

🔄 Limpeza de Dados

Data Cleaning

Detecção de Outliers: IQR method para remover valores anormais
Imputação: Forward fill para séries temporais faltando dados
Sincronização: Alinhamento temporal satélite + sensores
Validação: Verifica consistência entre fontes
🔬 Feature Engineering

Criação de Variáveis

Média Móvel: 7 e 14 dias (tendência)
Taxa Mudança: Variação temporal de cada variável
Índices Compostos: Combinações de sensores
Defasagens: Valores de dias anteriores (t-1, t-7, t-30)
📊 Normalização

Scaling & Normalization

StandardScaler: Média 0, desvio padrão 1
MinMaxScaler: Intervalo [0, 1]
RobustScaler: Resistente a outliers
Dimensionalidade: 40+ features → entrada dos modelos

🧠 3. MODELOS MACHINE LEARNING

500 Árvores de Decisão

Profundidade: 20-30 nós

Acurácia: 88-92%
Tempo Treino: 45 segundos
